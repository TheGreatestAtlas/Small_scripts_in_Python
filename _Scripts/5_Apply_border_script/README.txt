Skrypt służy do nakładania obramówek na bitmapki mapek do KnightShift. Skrypt jest o tyle intuicyjny, że zapisuje plik wynikowy do pliku .bmp o takich samych parametrach
co bitmapa wejściowa, czyli gra na spokojnie odczyta plik wyjściowy z tego skryptu bez żadnych problemów.
Podajemy ścieżkę do pliku z bitmapą, następnie podajemy ścieżkę do pliku z obramówką, a na sam koniec podajemy ścieżkę do pliku wynikowego.
Pamietajmy o wpisywaniu dokładnych formatów podczas podawania nazw plików...

Skrypt działa również w trybie argc&argv czyli podajemy odpowiednio nazwe bitmapy, obramówki i pliku wyjściowego.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
EN:

The script is used to apply images to bitmaps of maps for KnightShift. The script is intuitive insofar as it saves the resulting file to a .bmp file with the same parameters
as the input bitmap, that is, the game will calmly read the output file from this script without any problems.
We give the path to the bitmap file, then we give the path to the frame file, and at the very end we give the path to the resulting file.
Let's remember to type the exact formats when specifying file names....

The script also works in argc&argv mode, that is, you specify the name of the bitmap, frame and output file, respectively.
