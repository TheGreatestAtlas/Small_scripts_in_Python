Skrypt służy do nakładania obramówek na bitmapki mapek do KnightShift. Skrypt jest o tyle intuicyjny, że zapisuje plik wynikowy do pliku .bmp o takich samych parametrach
co bitmapa wejściowa, czyli gra na spokojnie odczyta plik wyjściowy z tego skryptu bez żadnych problemów.
Skrypt uruchamia interfejs do wyboru plików oraz zapisu.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
EN:

The script is used to apply borders to KnightShift map bitmaps. The script is intuitive in that it saves the resulting file to a .bmp file with the same parameters
as the input bitmap, meaning that the game will read the output file from this script without any problems.
The script launches an interface for selecting files and saving.
